Решение СЛАУ методом Гаусса

Не судите строго эту инструкцию :)

Для просмотра решения откройте файл GaussSolution.sln в MS Visual Studio
Решение состоит из основнйо формы "MainForm" и 2-х классов "Gauss" и "GaussStep".
Запустите приложение (Ctrl+F5), при этом откроется основная форма решения. 
На форме находятся:
- поле для указания количества неизвестных системы уравнения (от 2 до 10)
- Таблицы: "Матрица А" - таблица коэффициентов перед неизвестными, "Матрица B" - таблица свободных коэффициентов, "Матрица X" - таблица решений, "Матрица Е" - таблица погрешностей
- кнопки "<<" ">>" для анализа промежуточных результатов 
- подменю "Тест", которое содержит несколько различных вариантов для тестирования алгоритма.
- кнопка "Вычислить" - по нажатию кнопки собственно и производится попытка решить систему уравнений методом Гаусса

Собственно, пользователь должен указать количество неизвестных, заполнить матрицы А и В, и нажать кнопку "Выполнить". 
В матрице X будут отражены значения неизвестных (в порядке убывания x1,x2,...,xN), в столбцах матрицы E - погрешность для каждого уравнения системы.
Пользователю доступен анализ шагов алгоритма посредством кнопок ">>" и "<<".
